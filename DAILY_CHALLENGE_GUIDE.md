## Tips for Maintaining Streaks

1. **Check Daily** - Visit the game every day at midnight UTC
2. **Quick Challenges** - Some challenges take only minutes
3. **Plan Ahead** - Know which challenge type you're good at
4. **Practice Mode** - Practice challenge types in regular mode first
5. **Power-up Knowledge** - Learn which power-ups help each challenge

## Daily Challenge Leaderboard

### What's Tracked
- Player name
- Challenge type completed
- Completion time
- Streak at completion time

### How to Rank
- Faster completion times rank higher
- Only daily challenge runs count
- Top 20 scores displayed
- Leaderboard persists locally

## Challenge Reset Schedule

- **Daily Reset**: Midnight UTC
- **Streak Continue**: Complete challenge before next reset
- **Streak Break**: Miss one day = streak resets to 0
- **Completion**: Can only complete once per day for rewards

## Troubleshooting

### Challenge Not Updating?
- Check your system date/time
- Clear browser cache
- Verify localStorage is enabled

### Streak Not Counting?
- Complete the challenge fully before midnight UTC
- Check completion confirmation popup appeared
- Verify streak badge updates

### Progress Not Tracking?
- Make sure you started via Daily Challenge menu
- Don't reload page during challenge
- Complete challenge objectives in one run

## FAQ

**Q: Can I replay challenges for better times?**  
A: Yes, but rewards only given once per day

**Q: What happens if I miss a day?**  
A: Streak resets to 0, start building again tomorrow

**Q: Are challenges the same for everyone?**  
A: Yes! Same seed = same challenge globally

**Q: Can I do challenges in multiplayer?**  
A: Currently single-player only

**Q: Do power-ups help with challenges?**  
A: Yes! Shield helps Perfect Run, Sprint helps Speed Run, etc.

**Q: How are challenge parameters decided?**  
A: Generated from daily seed with controlled randomness
